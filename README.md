# IMBT 人格测试

一个基于 MBTI 理论的人格测试 PWA 应用，采用现代化 UI 设计。

## 功能特性

- **60 道测试题目** - 全面的 MBTI 人格评估
- **四个维度分析** - E/I、S/N、T/F、J/P 维度百分比展示
- **16 种人格类型** - 详细的人格描述、优势、适合职业和代表人物
- **多语言支持** - 支持简体中文、English、日本語三种语言
- **测试历史记录** - 保存历史测试结果，随时回顾
- **PWA 支持** - 可安装到桌面，离线可用
- **暗色模式** - 支持亮色/暗色主题切换
- **响应式设计** - 适配各种屏幕尺寸

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS
- React Router
- i18next (国际化)
- PWA (Service Worker)

## 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 代码检查
pnpm lint
pnpm typecheck
```
