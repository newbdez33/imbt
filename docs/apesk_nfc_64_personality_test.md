## 一、测试简介

### 什么是 APESK NFC？

**NFC** 全称 **NO-Forced Choice 16Personalities Type Scale**，是由中国才储（APESK）团队在经典 MBTI 理论基础上改进而来的人格测评工具。

**核心改进：**
- **非迫选模式**：不需要"非此即彼"，可以按程度选择倾向，甚至选择中立
- **6维度模型**：在 MBTI 原有4个维度基础上增加了第5维度（待人接物 H/C）和第6维度（果断性 A/O），将16种人格扩展为 **64种人格类型**
- **本土化适配**：基于近4000万中国用户采样数据优化

### 测试基本信息

| 项目 | 内容 |
|------|------|
| 题目数量 | 93题（STEP-II为333题） |
| 答题方式 | 每题5个等级选项（非迫选滑动条） |
| 建议时间 | 10分钟内完成 |
| 量表版本 | V2020（量表N） |

### 答题注意事项

1. 在10分钟内不假思索完成
2. 呈现**真实的自己**，而不是你期待的样子
3. 尽量避免"中立"的答案

## 二、六大维度说明

### 原有MBTI四维度

| 维度 | 极性A | 极性B | 含义 |
|------|-------|-------|------|
| **维度1：精力来源** | E（外向 Extraversion） | I（内向 Introversion） | 能量来源于外部世界还是内心世界 |
| **维度2：感知方式** | S（实感 Sensing） | N（直觉 iNtuition） | 获取信息偏重具体事实还是抽象概念 |
| **维度3：决策方式** | T（思考 Thinking） | F（情感 Feeling） | 做决定偏重逻辑分析还是情感价值 |
| **维度4：生活态度** | J（判断 Judging） | P（知觉 Perceiving） | 生活方式偏重计划有序还是灵活随性 |

### APESK新增两维度

| 维度 | 极性A | 极性B | 含义 |
|------|-------|-------|------|
| **维度5：待人接物** | H（温暖 Hospitable） | C（高冷 Chilly） | 人际交往中是温暖亲和还是保持距离 |
| **维度6：果断性** | A（果断 Aspiring） | O（纠结 Torn） | 问题解决中是果断进取还是犹豫不决 |

### 64种人格类型

四个MBTI维度组合产生16种基础类型，再乘以 H/C × A/O = 4种子类型：

> 16 × 4 = **64种人格类型**

例如：INTJ-H-A、INTJ-H-O、INTJ-C-A、INTJ-C-O

## 三、完整93题题目

### 答题方式说明

每题为滑动条形式，从左到右分为5个等级：

| 位置 | 含义 | 计分 |
|------|------|------|
| 最左（红色） | 完全符合选项A | A方向 +2 |
| 偏左 | 比较符合选项A | A方向 +1 |
| 中间（灰色） | 中立 | 0 |
| 偏右 | 比较符合选项B | B方向 +1 |
| 最右（绿色） | 完全符合选项B | B方向 +2 |

## 四、计分方法

### 4.1 NFC非迫选计分规则

与传统MBTI的二选一不同，NFC采用**5级量表**：

```
完全A ← 偏向A ← 中立 → 偏向B → 完全B
 +2       +1       0       +1       +2
```

每个维度分别统计两端的得分。

### 4.2 各维度计分步骤

#### 步骤一：统计各维度原始得分

将93题按所属维度归类，分别统计每个维度两端的累积得分：

| 维度 | 统计项 | 题目编号（大致分布） |
|------|--------|---------------------|
| E/I | E分 vs I分 | 3, 9, 17, 23, 25, 28, 32, 34, 37, 38, 43, 47, 51, 58, 62, 64, 66, 75, 80 等 |
| S/N | S分 vs N分 | 6, 10, 11, 20, 27, 29, 40, 45, 46, 49, 50, 53, 55, 60, 63, 67, 69, 72, 73, 76, 81, 82 等 |
| T/F | T分 vs F分 | 1, 4, 7, 13, 15, 19, 21, 24, 26, 30, 31, 35, 36, 48, 52, 54, 56, 78, 83 等 |
| J/P | J分 vs P分 | 2, 5, 8, 12, 14, 16, 18, 22, 33, 39, 41, 42, 44, 57, 59, 61, 65, 68, 70, 71, 74, 77, 79 等 |
| A/O | A分 vs O分 | 85-90 等 |
| H/C | H分 vs C分 | 91-93 及补充题 |

#### 步骤二：计算各维度倾向度

```
倾向度 = |一端得分 - 另一端得分|
倾向方向 = 得分较高的一端
```

例如：E维度得分 = 18，I维度得分 = 26
- 倾向方向：**I（内向）**
- 倾向度：26 - 18 = **8**

#### 步骤三：确定类型代码

根据6个维度各自的倾向方向，组合成最终类型代码。

**示例：**

| 维度 | 结果 |
|------|------|
| E/I | I（倾向度8） |
| S/N | N（倾向度12） |
| T/F | T（倾向度6） |
| J/P | J（倾向度10） |
| H/C | C（倾向度4） |
| A/O | A（倾向度7） |

→ 最终人格类型：**INTJ-C-A**

### 4.3 倾向度等级参考

| 倾向度范围 | 等级 | 含义 |
|-----------|------|------|
| 0-5 | 轻微 | 两端差异不明显，可能在不同情境下切换 |
| 6-15 | 中等 | 有明确的偏好倾向 |
| 16-25 | 明显 | 倾向非常显著 |
| 26+ | 极端 | 几乎完全偏向一端 |

### 4.4 关于中立答案

NFC允许选择中立（0分），这意味着：
- 某些维度可能出现**两端得分相同**的情况
- 此时该维度被标记为"中立"或"边界型"
- 例如：E/I维度都得15分 → 标记为 "ie中立人"
